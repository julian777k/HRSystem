## 1. 로그인 / 로그아웃

### 로그인

1. 웹 브라우저를 열고 시스템 주소로 접속합니다.
   - 사내 환경: `http://서버IP:3000` (담당자에게 문의)
   - 로컬 테스트: `http://localhost:3000`

2. 로그인 화면이 표시됩니다.

3. **이메일** 항목에 등록된 회사 이메일을 입력합니다.
   - 예: `hong@company.com`

4. **비밀번호** 항목에 비밀번호를 입력합니다.
   - 최초 로그인 시 관리자로부터 전달받은 초기 비밀번호를 사용합니다.

5. **"로그인"** 버튼을 클릭합니다.

6. 로그인에 성공하면 역할에 따라 화면으로 이동합니다.
   - **일반 직원**: 대시보드(`/dashboard`)
   - **관리자**: 관리자 콘솔(`/admin`)

> **참고**: 비밀번호를 잊은 경우 로그인 화면의 **"비밀번호를 잊으셨나요?"** 링크를 클릭하여 이메일로 재설정할 수 있습니다.

### 로그아웃

1. 화면 우측 상단의 **"로그아웃"** 버튼을 클릭합니다.
2. 로그인 화면으로 돌아갑니다.

> **보안 안내**: 공용 컴퓨터를 사용하는 경우 반드시 로그아웃하세요. 일정 시간(1시간) 동안 활동이 없으면 자동으로 로그아웃됩니다.

## 2. 화면 구성

MSA HR 시스템은 크게 세 가지 영역으로 구성됩니다.

### 상단 헤더

- **MSA HR 로고**: 클릭하면 대시보드로 이동합니다.
- **사용자 이름**: 현재 로그인한 사용자 정보가 표시됩니다.
- **로그아웃 버튼**: 시스템에서 로그아웃합니다.

### 좌측 사이드바

시스템의 모든 메뉴가 카테고리별로 정리되어 있습니다.

**휴가 메뉴**:

| 메뉴 | 설명 |
|------|------|
| 나의 휴가 | 내 연차 잔여일수, 시간 지갑 잔액, 휴가 신청 내역 확인 |
| 휴가사용현황 | 부서 또는 전사 휴가 사용 현황 (캘린더/목록 뷰) |
| 휴가신청관리 | 내가 결재해야 할 휴가 신청 목록 (결재자 전용) |
| 휴가관리대장 | 전체 직원의 휴가 기록 조회 (관리자 전용) |
| 휴가부여 | 연차 수동 부여/차감 (관리자 전용) |

**복지 메뉴**:

| 메뉴 | 설명 |
|------|------|
| 복지혜택 | 회사에서 제공하는 복지 항목 카탈로그 |
| 복지신청 | 복지 혜택 신청 및 내 신청 내역 확인 |
| 복지관리 | 복지 카테고리/항목 관리 (관리자 전용) |

**근태관리 메뉴**:

| 메뉴 | 설명 |
|------|------|
| 오늘 근무 | 출근/퇴근 시간 기록, 현재 근무 상태 확인 |
| 내 근태현황 | 월별 출퇴근 기록 조회, 근무시간 요약 |
| 연장근무 신청 | 평일야간/주말/공휴일 연장근무 신청 |
| 연장근무 현황 | 부서원 연장근무 승인/반려 (관리자 전용) |

**기본설정 메뉴** (관리자 전용):

| 메뉴 | 설명 |
|------|------|
| 권한/결재선 설정 | 결재 단계 및 결재자 설정 |
| 휴가규정 관리 | 연차 규정 설정 |
| 보상정책 설정 | 시간외근무 보상 배율, 차감 순서, 시간 단위 설정 |
| 시간외근무 설정 | 시간외근무 정책 및 신청 승인/반려 |
| 직원관리 | 직원 등록/수정/삭제 |
| 복지관리 | 복지 카테고리 및 항목 CRUD |
| 외부서비스 연동 | 구글 캘린더 등 연동 설정 |

> **참고**: 권한에 따라 표시되는 메뉴가 다릅니다. 사이드바 메뉴 그룹은 클릭하여 접기/펼치기가 가능합니다.

### 메인 콘텐츠 영역

좌측 사이드바에서 선택한 메뉴의 내용이 표시되는 중앙 영역입니다.

## 3. 대시보드

로그인 후 가장 먼저 표시되는 화면입니다.

대시보드에서 확인할 수 있는 정보:

- **잔여 연차**: 현재 사용 가능한 연차 일수
- **사용한 연차**: 올해 사용한 연차 일수
- **결재 대기**: 내가 승인/반려해야 할 건수 (결재자인 경우)
- **나의 최근 신청**: 최근 제출한 휴가/시간외근무 신청 내역
- **오늘의 부재자**: 오늘 휴가 중인 같은 부서 직원 목록

## 4. 나의 휴가

좌측 메뉴에서 **"나의 휴가"** 를 클릭하면 개인 휴가 현황 페이지가 표시됩니다.

### 연차 잔여일수 확인

페이지 상단에 올해의 연차 현황이 카드 형태로 표시됩니다.

| 항목 | 설명 |
|------|------|
| 총 부여일수 | 올해 부여된 전체 연차 일수 |
| 사용일수 | 이미 사용한(승인 완료된) 연차 일수 |
| 잔여일수 | 현재 사용 가능한 연차 일수 (총 부여 - 사용) |

### 휴가 신청 내역

페이지 하단에 본인이 신청한 휴가 목록이 표시됩니다.

| 항목 | 설명 |
|------|------|
| 휴가 종류 | 연차, 병가, 경조사 등 |
| 기간 | 휴가 시작일 ~ 종료일 |
| 일수 | 사용 일수 (0.5일, 1일 등) |
| 상태 | 대기중 / 진행중 / 승인 / 반려 / 취소 |
| 사유 | 휴가 신청 사유 |

**상태 설명**:

| 상태 | 의미 |
|------|------|
| 대기중 (PENDING) | 결재자의 승인을 기다리는 중 |
| 진행중 (IN_PROGRESS) | 다단계 결재 중 (일부 승인 완료) |
| 승인 (APPROVED) | 모든 결재자가 승인 완료 |
| 반려 (REJECTED) | 결재자가 반려한 상태 |
| 취소 (CANCELLED) | 본인이 취소한 상태 |

## 5. 휴가 신청

### 신청 절차

1. 좌측 메뉴에서 **"나의 휴가"** 를 클릭합니다.

2. 페이지 상단의 **"휴가 신청"** 버튼을 클릭합니다.

3. 휴가 신청 양식이 표시됩니다. 아래 항목을 입력합니다.

#### 휴가 종류 선택

드롭다운에서 원하는 휴가 종류를 선택합니다.

| 휴가 종류 | 설명 | 연차 차감 |
|-----------|------|-----------|
| 연차휴가 | 일반 연차 사용 | 차감됨 |
| 병가 | 질병/부상으로 인한 휴가 (진단서 필요) | 차감 안 됨 |
| 경조사 | 결혼, 장례 등 경조사 | 차감 안 됨 |
| 출산휴가 | 출산 전후 휴가 (최대 90일) | 차감 안 됨 |
| 배우자출산 | 배우자 출산 시 (최대 10일) | 차감 안 됨 |
| 공가 | 공적 의무 수행 (예비군, 투표 등) | 차감 안 됨 |
| 특별휴가 | 회사 지정 특별 휴가 | 차감 안 됨 |

#### 휴가 단위 선택

| 단위 | 설명 |
|------|------|
| 종일 | 하루 전체 (1일) |
| 오전 반차 | 오전만 휴가 (0.5일), 오후 출근 |
| 오후 반차 | 오후만 휴가 (0.5일), 오전 근무 후 퇴근 |
| 시간 단위 | 시간 단위로 사용 |

#### 날짜 선택

- **시작일**: 휴가 시작 날짜를 달력에서 선택합니다.
- **종료일**: 휴가 종료 날짜를 선택합니다. 하루만 사용하는 경우 시작일과 동일하게 선택합니다.

> **참고**: 주말과 공휴일은 자동으로 제외되어 계산됩니다.

#### 사유 입력

휴가 사유를 입력합니다. 필수 입력은 아니지만, 결재자가 참고할 수 있도록 간단히 기재하는 것이 좋습니다.

- 예: "개인 사유", "가족 행사", "병원 진료" 등

4. 모든 항목을 입력한 후 **"신청"** 버튼을 클릭합니다.

5. "휴가 신청이 완료되었습니다" 메시지가 표시되면 정상적으로 제출된 것입니다.

6. 결재자에게 자동으로 알림이 발송됩니다.

### 신청 시 주의사항

- 잔여 연차가 부족하면 연차휴가를 신청할 수 없습니다.
- 이미 승인된 휴가와 날짜가 겹치는 경우 신청할 수 없습니다.
- 병가의 경우 진단서 등 관련 서류 제출이 필요할 수 있습니다.
- 과거 날짜로의 휴가 신청은 관리자 승인이 필요합니다.

## 6. 휴가 취소

### 취소 가능 조건

- **대기중 (PENDING)** 또는 **진행중 (IN_PROGRESS)** 상태의 휴가만 취소할 수 있습니다.
- **승인 완료 (APPROVED)** 된 휴가는 관리자에게 취소를 요청해야 합니다.
- **이미 시작된 휴가**는 취소할 수 없습니다.

### 취소 절차

1. 좌측 메뉴에서 **"나의 휴가"** 를 클릭합니다.

2. 휴가 신청 내역에서 취소하려는 신청 건을 찾습니다.

3. 해당 행의 **"취소"** 버튼을 클릭합니다.

4. 취소 사유를 입력합니다.
   - 예: "일정 변경", "사유 소멸" 등

5. **"확인"** 버튼을 클릭합니다.

6. 휴가 상태가 **"취소"** 로 변경됩니다.
   - 연차휴가인 경우, 차감되었던 연차가 자동으로 복원됩니다.

## 7. 결재 처리 (승인/반려)

> 이 기능은 결재 권한이 있는 직원(부서장, 팀장 등)만 사용할 수 있습니다.

### 결재 대기 목록 확인

1. 좌측 메뉴에서 **"휴가신청관리"** 를 클릭합니다.

2. 내가 결재해야 할 신청 목록이 표시됩니다.

각 신청 건에는 다음 정보가 표시됩니다:

| 항목 | 설명 |
|------|------|
| 신청자 | 휴가를 신청한 직원 이름 |
| 부서 | 신청자의 소속 부서 |
| 휴가 종류 | 연차, 병가 등 |
| 기간 | 휴가 시작일 ~ 종료일 |
| 일수 | 사용 일수 |
| 사유 | 신청 사유 |
| 상태 | 현재 결재 상태 |

### 승인 처리

1. 승인할 신청 건을 클릭하여 상세 내용을 확인합니다.

2. 내용을 검토한 후 **"승인"** 버튼을 클릭합니다.

3. 필요한 경우 코멘트를 입력할 수 있습니다.
   - 예: "승인합니다", "좋은 휴가 보내세요" 등

4. **"확인"** 을 클릭하면 승인이 완료됩니다.

- **1단계 결재**: 즉시 최종 승인 처리
- **다단계 결재**: 다음 결재자에게 자동 전달

### 반려 처리

1. 반려할 신청 건을 클릭하여 상세 내용을 확인합니다.

2. **"반려"** 버튼을 클릭합니다.

3. **반려 사유를 반드시 입력합니다.**
   - 예: "해당 기간 중요 회의가 예정되어 있습니다. 일정 조정 부탁드립니다."
   - 반려 사유를 입력하지 않으면 반려가 되지 않습니다.

4. **"확인"** 을 클릭하면 반려가 완료됩니다.

5. 신청자에게 반려 알림이 발송됩니다.

### 결재 이력 확인

처리 완료된 결재 건은 "처리 완료" 탭에서 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 처리 일시 | 승인/반려한 날짜 및 시간 |
| 처리 결과 | 승인 / 반려 |
| 코멘트 | 처리 시 입력한 코멘트 |

## 8. 휴가 사용 현황 조회

1. 좌측 메뉴에서 **"휴가사용현황"** 을 클릭합니다.

2. 두 가지 뷰로 조회할 수 있습니다.

### 캘린더 뷰 (기본)

월간 캘린더 형태로 직원들의 휴가 현황을 시각적으로 확인합니다.

- **왼쪽 패널**: 부서 트리 - 부서를 클릭하면 해당 부서원의 휴가만 필터링
- **캘린더**: 날짜별로 휴가 사용 직원이 색상 뱃지로 표시
  - 연차: 초록색, 반차: 하늘색, 경조: 주황색, 병가: 노란색
  - 공휴일은 빨간색 텍스트로 표시
  - 토/일요일은 연한 노란색 배경
- **월 이동**: 이전/다음 월 버튼으로 이동

### 목록 뷰

표 형태로 직원별 휴가 사용률을 확인합니다.

| 항목 | 설명 |
|------|------|
| 직원명 | 직원 이름 |
| 부서 | 소속 부서 |
| 총 부여 | 올해 부여된 연차 |
| 사용 | 사용한 연차 |

| 항목 | 설명 |

| 근무 시간 | 해당일 실근무 시간 |
| 상태 | 정상 / 지각 / 조퇴 / 결근 |

4. 결재자에게 승인 요청이 전달됩니다.

## 10. 시간외근무 신청

근무시간 외에 추가 근무가 필요한 경우 시간외근무를 신청합니다.

### 시간외근무 유형

| 유형 | 설명 | 수당 배율 |
|------|------|-----------|
| 평일 야간 | 평일 22:00 ~ 06:00 사이 근무 | 1.5배 |
| 주말 | 토요일, 일요일 근무 | 1.5배 |
| 공휴일 | 법정 공휴일 근무 | 2.0배 |

### 신청 절차

1. 대시보드 또는 해당 메뉴에서 **"시간외근무 신청"** 을 클릭합니다.

2. 시간외근무 신청 양식에서 아래 항목을 입력합니다.

| 항목 | 설명 | 예시 |
|------|------|------|
| 근무 일자 | 시간외근무를 하는 날짜 | 2026-02-14 |
| 근무 유형 | 평일 야간 / 주말 / 공휴일 | 주말 |
| 시작 시간 | 시간외근무 시작 시간 | 09:00 |
| 종료 시간 | 시간외근무 종료 시간 | 18:00 |
| 사유 | 시간외근무 사유 (필수) | 프로젝트 마감 |

3. **"신청"** 버튼을 클릭합니다.

4. 결재자에게 승인 요청이 전달됩니다.

### 주의사항

- 주당 최대 시간외근무 시간은 **12시간**입니다 (기본 설정).
- 월간 최대 시간외근무 시간은 **52시간**입니다 (기본 설정).
- 한도를 초과하는 신청은 시스템에서 자동으로 제한될 수 있습니다.
- 시간외근무는 사전 신청이 원칙이며, 사후 신청은 관리자 승인이 필요합니다.

## 11. 시간 지갑 (보상시간/연차시간)

시간 지갑은 보상시간과 연차시간을 시간 단위로 통합 관리하는 시스템입니다.

### 시간 지갑 확인

대시보드와 나의 휴가 페이지에서 시간 지갑 잔액을 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 보상시간 잔액 | 시간외근무로 적립된 보상시간 (예: 4.5h) |
| 연차시간 잔액 | 연차를 시간으로 환산한 잔액 (예: 120h) |
| 총 잔여 시간 | 보상시간 + 연차시간 합계 |
| 총 잔여 일수 | 총 잔여 시간 ÷ 1일 근무시간 |

### 보상시간 적립

시간외근무가 승인되면 보상정책 배율에 따라 보상시간이 자동 적립됩니다.

예시: 평일 3시간 연장근무 × 1.5배 = **4.5시간 보상시간 적립**

### 휴가 사용 시 자동 차감

휴가가 승인되면 관리자가 설정한 차감 순서에 따라 자동으로 차감됩니다.

- **기본 순서**: 보상시간 → 연차시간
- 1일 휴가 = 8시간 차감 (관리자 설정에 따라 변동)
- 반차 = 4시간 차감

## 12. 복지혜택 조회 및 신청

### 복지혜택 조회

좌측 메뉴에서 **"복지혜택"** 을 클릭하면 회사에서 제공하는 복지 항목을 카탈로그 형태로 볼 수 있습니다.

3. **"신청"** 버튼을 클릭합니다.
4. 관리자가 승인하면 혜택이 적용됩니다.

### 신청 내역 확인

- 대기 중인 신청은 취소 가능

## 14. 비밀번호 변경 / 재설정

개인 비밀번호를 변경하는 방법입니다.

### 변경 절차

1. 화면 우측 상단의 **사용자 이름** 영역을 클릭합니다.

2. 드롭다운 메뉴에서 **"비밀번호 변경"** 을 선택합니다.

3. 비밀번호 변경 양식에서 아래 항목을 입력합니다.

| 항목 | 설명 |
|------|------|
| 현재 비밀번호 | 지금 사용 중인 비밀번호 |
| 새 비밀번호 | 변경할 새 비밀번호 (8자 이상) |
| 새 비밀번호 확인 | 새 비밀번호 재입력 |

4. **"변경"** 버튼을 클릭합니다.

5. "비밀번호가 변경되었습니다" 메시지가 표시되면 완료입니다.

### 비밀번호 규칙

- 최소 **8자 이상** 이어야 합니다.
- 이전 비밀번호와 동일한 비밀번호는 사용할 수 없습니다.
- 보안을 위해 주기적으로 비밀번호를 변경하는 것을 권장합니다.

---

### 비밀번호 재설정 (이메일)

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 링크를 클릭합니다.
2. 등록된 이메일 주소를 입력합니다.
3. 이메일로 전송된 재설정 링크를 클릭합니다.
4. 새 비밀번호를 입력하고 변경합니다.

## 15. 자주 묻는 질문

### Q1. 내 연차가 몇 일 남았는지 어떻게 확인하나요?

좌측 메뉴에서 **"나의 휴가"** 를 클릭하면 페이지 상단에 잔여 연차가 표시됩니다.

### Q2. 반차를 사용하면 연차가 0.5일 차감되나요?

네, 오전 반차 또는 오후 반차를 사용하면 연차가 **0.5일** 차감됩니다.

### Q3. 휴가를 신청했는데 상태가 "대기중"이에요. 언제 승인되나요?

"대기중"은 결재자가 아직 처리하지 않은 상태입니다. 결재자(보통 직속 상관 또는 부서장)에게 확인을 요청하세요. 급한 경우 결재자에게 직접 연락하시기 바랍니다.

### Q4. 이미 승인된 휴가를 취소할 수 있나요?

승인 완료된 휴가는 본인이 직접 취소할 수 없습니다. 관리자(인사담당자)에게 취소를 요청하세요.

### Q5. 주말을 포함해서 휴가를 신청하면 주말도 연차가 차감되나요?

아니요, 주말(토요일, 일요일)은 자동으로 제외되어 계산됩니다. 예를 들어 금요일~월요일(4일)로 신청해도 실제 차감되는 연차는 2일(금요일, 월요일)입니다.

### Q6. 휴가 종류별로 사용 가능한 일수가 정해져 있나요?

- **연차휴가**: 근속연수에 따라 자동 부여됩니다 (근로기준법 기준).
- **병가**: 별도 한도 없음 (회사 규정에 따름).
- **출산휴가**: 최대 90일.
- **배우자출산휴가**: 최대 10일.
- 기타 휴가는 관리자가 설정한 규정에 따릅니다.

### Q7. 시간외근무를 했는데 수당은 어떻게 계산되나요?

시간외근무 수당은 시스템에 설정된 배율에 따라 계산됩니다 (기본: 평일 야간/주말 1.5배, 공휴일 2.0배). 구체적인 수당 금액은 인사담당자에게 문의하세요.

### Q8. 로그인이 안 돼요.

- 이메일 주소가 정확한지 확인하세요.
- 비밀번호의 대소문자를 확인하세요.
- Caps Lock이 켜져 있지 않은지 확인하세요.
- 여러 번 실패하면 잠시 후 다시 시도하세요.
- 그래도 안 되면 관리자에게 비밀번호 초기화를 요청하세요.

### Q9. 화면이 이상하게 보여요.

- 브라우저를 최신 버전으로 업데이트하세요.
- Chrome, Safari, Edge 브라우저를 권장합니다.
- Internet Explorer(IE)는 지원하지 않습니다.
- `Ctrl + Shift + R` (Mac: `Cmd + Shift + R`)로 강제 새로고침을 시도하세요.

### Q10. 모바일에서도 사용할 수 있나요?

MSA HR은 반응형 웹으로 제작되어 모바일 브라우저에서도 사용할 수 있습니다. 하지만 PC 환경에서 사용하는 것을 권장합니다.
