CHAPTER 1

시장 분석

1-1. 타겟 시장

구분내용
1차 타겟직원 5~50명 중소기업, 스타트업 (엑셀로 인사관리 중, SaaS 구독료 부담)
2차 타겟직원 50~100명 중견기업 (플렉스·시프티 구독 중, 비용 절감 욕구)
3차 타겟비영리·학원·소규모 법인 (저렴한 HR 솔루션 필요)

1-2. 시장 규모

  • 한국 중소기업 수: 약 400만 개 (전체 사업체의 99.9%)
  • 10~100명 규모 기업: 약 30만 개
  • 그 중 전문 HR 솔루션 미사용 추정: 80% 이상 (엑셀·수기 관리)
  • HR SaaS 시장 이탈 수요: SaaS 구독 후 1년 내 이탈률 약 30~40%
핵심 인사이트 HR SaaS 시장에는 두 가지 불만족 고객군이 존재한다:
① "엑셀은 한계인데 SaaS는 비싸다" — 미진입 고객
② "SaaS 쓰고 있는데 매달 나가는 돈이 아깝다" — 이탈 예정 고객
KeystoneHR의 "1회 구매 10년 사용" 모델은 이 두 고객군 모두를 타겟한다.

1-3. 크몽 플랫폼 현황

항목수치
IT·프로그래밍 카테고리 전체 서비스~53,000건
"HR 시스템" 검색 결과매우 적음 (10건 미만)
클라우드 HR SaaS 1회 구매0건 (사실상 공백)
블루 오션 크몽에서 "구독료 없는 HR SaaS" 카테고리는 완전한 공백 상태. 설치형도, SaaS형도 아닌 "1회 구매 클라우드 HR"이라는 새로운 카테고리를 개척한다.

3-2. 핵심 차별화 포인트

1. SaaS인데 구독료 0원
업계 최초. 클라우드 HR인데 월 비용이 없다. "SaaS의 편리함 + 설치형의 가격"이라는 새 카테고리.
2. 1회 구매 10년 사용
49만원 한 번이면 10년간 사용. 연 4.9만원 = 월 4,083원. 시프티 3일치 비용.
3. 설치 불필요, 즉시 시작
가입 후 회사명.keystonehr.app 즉시 발급. exe 설치, 서버 세팅 전혀 없음. 5분 만에 시작.
4. 인원 제한 없는 기본 50명
기본 50명 포함. 추가 인원 확장 저렴. SaaS처럼 인당 과금이 늘어나지 않음.
5. 복리후생 풀 워크플로우 (독점 기능)
시프티·플렉스에 없는 기능. 경조사비, 자기개발비 등 복지 항목 자유 생성→신청→결재→현황. 예산 추적, 자동 승인 옵션까지 풀 워크플로우 제공.
6. 24/7 무중단 운영
Cloudflare 글로벌 엣지 기반. PC를 켜둘 필요 없음. 전 세계 어디서든 접속 가능.
7. 자동 업데이트
새 기능·버그 수정이 자동 반영. 파일 교체나 재설치 없음.
8. 서비스 종속 없음
데이터 내보내기(Excel/CSV) 언제든 가능. SaaS 벤더 종속 없이 데이터 주권 보장.
9. 메신저 연동 — Slack/카카오워크/Teams (독점 기능)
웹훅 기반 Slack, 카카오워크, Microsoft Teams 연동 내장. 일간/주간 휴무 현황을 팀 채널에 자동 발송. 경쟁사는 별도 요금 또는 외부 연동 필요.
10. 엔터프라이즈급 보안
사용자/관리자 JWT 이중 분리(토큰 재사용 공격 차단), 최초 로그인 시 비밀번호 강제 변경, Rate Limiting, 비밀번호 최대 길이 제한, 보안 헤더 적용.

CHAPTER 4 — NEW

트롤링 마케팅 전략

전략 핵심 경쟁사의 "월 N,000원"이라는 가격 표기를 역이용한다.
고객은 "월 7,000원"이 싸다고 느끼지만, 50명 × 12개월 × 10년 = 수천만원이라는 사실을 모른다.
이 "인지 격차"를 폭로하는 것이 트롤링 마케팅의 핵심이다.

4-1. 경쟁사별 트롤링 포인트

vs 플렉스 (flex) — "인당 7,000원"의 진실
항목플렉스가 말하는 것플렉스가 말하지 않는 것
가격 "인당 월 7,000원" 기본료 7만원 + 인당 7,000원 = 50명이면 월 42만원
연 비용 표시 안 함 504만원/년
3년 비용 표시 안 함 1,512만원
10년 비용 표시 안 함 5,040만원
해지하면? "언제든 해지 가능" 해지하면 데이터 접근 불가. 지금까지 낸 돈 = 매몰 비용
메신저 연동 "연동 가능" (별도 설정) 별도 연동 필요. KeystoneHR은 Slack/카카오워크/Teams 웹훅이 기본 내장, 휴무 현황 자동 발송까지 무료

→ 마케팅 카피: "플렉스 '인당 7,000원'의 정체: 50명이면 연 504만원. 10년이면 5,040만원. 그리고 해지하면 0원으로 돌아갑니다. 메신저 연동? 그것도 별도입니다."

vs 시프티 (Shiftee) — "2,000원부터"의 함정
항목시프티가 말하는 것시프티가 말하지 않는 것
가격 "인당 월 2,000원~" 필수 기능 쓰려면 4,000원 플랜 필요. 50명 = 월 20만원
연 비용 표시 안 함 240만원/년 (상위 플랜 기준)
직원 증가 시 "합리적 인당 과금" 직원 100명 되면 월 40만원 → 연 480만원으로 2배
복리후생 "파트너 연동" 별도 서비스 가입 + 추가 비용 필요
메신저 연동 "추가 요금으로 제공" Slack/카카오워크/Teams 연동하려면 별도 과금. KeystoneHR은 기본 내장

→ 마케팅 카피: "시프티 '2,000원부터'? 실제로 50명이 쓰면 월 20만원. Slack 연동은 또 별도 과금. KeystoneHR은 49만원이면 10년, 메신저 연동까지 포함."

vs 원티드스페이스 — "올인원"의 가격표
항목원티드스페이스KeystoneHR
월 비용 (50명) ~30만원 0원 (1회 구매 완료)
3년 누적 ~1,080만원 49만원
기능 범위 HR + 채용 (하지만 채용이 필요 없는 회사에게는 과잉) HR 핵심 기능 집중 (필요한 것만)
메신저 연동 미제공 Slack/카카오워크/Teams 웹훅 내장 + 휴무 현황 자동 발송
복리후생 관리 미제공 예산 추적, 자동 승인까지 풀 워크플로우

CHAPTER 11

리스크 & 대응 전략

리스크영향대응
Cloudflare 요금 인상 높음 현재 tenant당 월 ~₩800. 5배 인상(₩4,000)에도 10년 ₩48만 = 수익 유지 가능. 이용약관에 불가항력 조항.
SaaS 대비 기능 부족 인식 중 "핵심 기능에 집중" 포지셔닝 + 복리후생 풀 워크플로우·메신저 연동(Slack/카카오워크/Teams)·엔터프라이즈급 보안 등 독점 기능 강조. 무료 체험으로 사전 검증.
크몽 초기 노출 부족 높음 Early Bird + 트롤링 블로그 SEO + 무료 체험 퍼널. 크몽 외 직접 유입 병행.
악의적 리뷰 / 환불 분쟁 중 14일 무료 체험 의무화 → "충분히 써보고 구매"를 강조. 환불 정책 명확 기재.
"10년 후 어떻게 되나요?" 우려 중 FAQ에 명시: "10년 후 ₩9.9만원에 갱신 가능. 갱신하지 않으면 데이터 내보내기 후 90일간 보관."
경쟁사의 가격 인하 대응 낮음 SaaS 구독 모델 자체가 월 과금이므로, 1회 구매 모델과 직접 가격 경쟁 어려움. 우리의 모델 자체가 방어막.
서비스 장애/다운타임 높음 Cloudflare Workers = 전 세계 엣지 배포, 단일 서버 장애 없음. SLA "목표 99%"로 법적 리스크 최소화.
번들 사이즈 한계 (3MB) 중 현재 3,059KB/3,072KB. 기능 추가 시 코드 최적화 또는 Workers 분리 배포 필요.
