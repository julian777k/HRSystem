주요 기능

📋
휴가 관리
연차·병가·경조 등 다양한 휴가 유형 설정. 신청→결재→잔여일수 자동 계산. 연도 이월, 자동 부여.
⏰
연장근무 관리
야근·휴일근무 신청 및 승인. 일/주/월 상한 설정. 보상휴가 자동 전환 (Time Wallet).
✅
전자 결재
다단계 결재 라인 설정. 부서장→팀장→대표 순서 자동 라우팅. 모바일에서도 승인 가능.
👥
직원 관리
직원 등록·수정·퇴사 처리. 부서·직급 조직도. 엑셀 일괄 업로드/다운로드.
🎁
복리후생 커스텀 독점 기능
단순 목록이 아닌 풀 워크플로우: 카테고리 생성 → 항목 관리 → 직원 신청 → 결재 → 사용 추적까지 한 곳에서. 항목별 예산 모니터링, 특정 카테고리 자동승인 설정 가능. 경조사비·자기개발비·건강검진·식대 등 원하는 복지 항목을 자유롭게 생성. 유명 HR SaaS에서는 별도 서비스 연동이 필요한 기능이 기본 탑재.
📊
대시보드
출근 현황, 휴가 통계, 부서별 인원, 결재 현황 한눈에. 실시간 업데이트.
🔔
알림 시스템
결재 요청·승인·반려 실시간 알림. 로그인하면 바로 확인. Webhook 연동으로 Slack·카카오워크·Teams에 자동 전송까지.
💬
메신저 연동 NEW
Slack, 카카오워크, Microsoft Teams에 일간/주간 휴가 현황을 자동 발송. "오늘 누가 휴가인지 일일이 확인"할 필요 없이 시스템이 팀 메신저에 자동 알림. 경쟁 HR SaaS는 별도 과금하거나 아예 미제공하는 기능이 기본 탑재.
🔐
보안 & 권한
엔터프라이즈급 보안: JWT 시크릿 분리, 강제 비밀번호 변경, Rate Limiting, 비밀번호 정책 적용. Cloudflare DDoS 방어 + WAF. 보안 헤더(CSP, HSTS, X-Frame-Options) 기본 적용. 역할별 접근 제어(관리자/부서장/일반), 감사 로그 자동 기록.
🏖️
공휴일 관리
연도별 공휴일 관리. 대체공휴일, 회사 자체 휴일 설정.

5. 자주 묻는 질문 (FAQ)

정말 추가 비용이 없나요?
네. 49만원 결제 후 10년간 모든 기능을 사용할 수 있습니다. 월 구독료, 서버비, 인당 추가 과금 모두 없습니다. 직원 50명 이내라면 추가 비용은 일절 발생하지 않습니다.
SaaS(클라우드)인데 왜 구독료가 없나요?
초기 50사 한정 "파운더스 플랜" 혜택입니다. 서비스 초기에 함께해주시는 기업에게 감사의 의미로 1회 결제 10년 사용 조건을 드리고 있습니다. 50사가 채워지면 이 혜택은 종료되며, 이후에는 월 구독 모델로 전환됩니다. 파운더스 플랜 고객은 10년간 기존 조건이 그대로 유지됩니다.
직원이 50명을 넘으면 어떻게 하나요?
100명 확장(+15만원) 또는 무제한 확장(+30만원) 패키지를 추가 구매하시면 됩니다. 1회 결제이며, 인당 과금이 아니라 확장 1번이면 끝입니다.
10년 후에는 어떻게 되나요?
10년 후 ₩99,000에 10년 갱신이 가능합니다. 갱신하지 않으실 경우, 데이터 내보내기(Excel/CSV) 기간 90일을 드린 후 계정이 종료됩니다. 데이터는 언제든 내보내실 수 있으므로 걱정하지 않으셔도 됩니다.
설치가 필요한가요? IT 담당자가 없는데 괜찮을까요?
설치가 전혀 필요 없습니다. 브라우저에서 주소(회사명.keystonehr.app)를 입력하면 바로 접속됩니다. 초기 설정도 마법사가 안내하므로 IT 담당자 없이 대표나 경영지원 담당자가 직접 운영할 수 있습니다.
모바일에서도 되나요?
네. 반응형 웹 디자인으로 PC, 스마트폰, 태블릿 어디서든 브라우저 접속만으로 사용 가능합니다. 별도 앱 설치 없이 동일한 기능을 사용할 수 있습니다.
데이터가 안전한가요?
Cloudflare의 글로벌 보안 인프라에서 운영됩니다. HTTPS 암호화, DDoS 방어가 기본 포함되어 있습니다. 역할별 접근 제어, 감사 로그 자동 기록 등 엔터프라이즈급 보안 기능도 탑재되어 있습니다.
데이터를 내보낼 수 있나요?
네. 언제든 Excel/CSV로 데이터를 내보내실 수 있습니다. 직원 정보, 휴가 내역, 근태 기록 등 모든 데이터를 다운로드할 수 있어 데이터 종속(vendor lock-in)이 없습니다.
서비스가 중단되면 어떻게 되나요?
이용약관에 따라 서비스 종료 시 최소 6개월 전 사전 고지를 드리며, 90일간 데이터 내보내기 기간을 보장합니다. 또한 인프라 비용이 매우 낮아 서비스 유지의 경제적 부담이 거의 없으므로 안정적 운영이 가능합니다.
기존 SaaS에서 데이터를 가져올 수 있나요?
Excel 파일로 직원 정보를 일괄 업로드할 수 있습니다. 기존 SaaS에서 데이터를 Excel로 내보내신 후, KeystoneHR에 업로드하시면 됩니다.
구매 전에 먼저 써볼 수 있나요?
네! 14일 무료 체험을 제공합니다. 크몽 메시지로 "무료 체험"이라고 보내주시면 즉시 체험 계정을 발급해드립니다. 신용카드 등록 없이, 모든 기능을 14일간 자유롭게 사용해보신 후 결정하실 수 있습니다.
