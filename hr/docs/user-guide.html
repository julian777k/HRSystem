1. 시스템 소개

HR SYSTEM은 중소기업을 위한 올인원 인사관리 솔루션입니다. 별도의 서버 구축이나 클라우드 비용 없이, 사내 PC 한 대에 설치하면 전 직원이 브라우저로 접속하여 사용할 수 있습니다.

주요 특징

  • 서버리스 설치 — 별도 서버 없이 PC 한 대에서 서버 + DB 자동 실행
  • 브라우저 접속 — 같은 네트워크 내 모든 기기에서 브라우저로 접속
  • 모바일 지원 — 스마트폰, 태블릿에서도 반응형 화면 제공
  • 휴가 관리 — 연차, 반차, 병가, 경조 등 다양한 휴가 유형 관리
  • 복지 관리 — 건강검진, 교육비, 경조금 등 복지 항목 신청/승인
  • 전자 결재 — 직급 기반 순차/선승인 결재 흐름
  • 출퇴근 관리 — 출퇴근 기록, 근태현황 조회, 연장근무 신청/승인
  • 데이터 내보내기 — Excel 형식으로 데이터 다운로드

3. 설치 가이드

중요 앱은 서버 역할을 하는 PC 한 대에만 설치합니다. 나머지 직원들은 브라우저로 접속합니다.

3.1 Windows 설치

1
제공된 HRInstall-0.2.0-x64.exe 파일을 실행합니다.
2
설치가 자동으로 진행됩니다. 완료 후 바탕화면에 HR SYSTEM 바로가기가 생성됩니다.
3
바로가기를 더블클릭하여 실행합니다. 처음 실행 시 초기 설정 화면이 나타납니다.
Windows 방화벽 알림 처음 실행 시 방화벽 허용 팝업이 나타날 수 있습니다. "액세스 허용"을 선택해야 다른 기기에서 접속이 가능합니다.

3.2 macOS 설치

1
제공된 HRInstall-0.2.0-arm64.dmg (M시리즈) 또는 HRInstall-0.2.0-x64.dmg (Intel) 파일을 실행합니다.
2
열린 창에서 HR SYSTEM 아이콘을 Applications 폴더로 드래그합니다.
3
Applications에서 HR SYSTEM을 실행합니다.
macOS 보안 알림 "확인되지 않은 개발자" 경고가 나타나면: 시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기"를 클릭합니다.

4. 초기 설정

앱을 처음 실행하면 초기 설정 마법사가 시작됩니다.

1
회사 정보 입력
회사명, 사업자번호, 대표자명을 입력합니다.
2
관리자 계정 생성
시스템 관리자의 이름, 이메일, 비밀번호를 설정합니다. 이 계정으로 모든 설정을 관리합니다.
3
기본 데이터 설정
부서, 직급, 휴가 유형 등 기본 데이터가 자동 생성됩니다.
4
설정 완료
대시보드로 이동합니다. 이제 관리자 메뉴에서 세부 설정을 진행합니다.

5. 접속 방법

5.1 서버 PC에서 접속

앱을 실행하면 자동으로 HR SYSTEM이 표시됩니다. 별도 조작이 필요 없습니다.

5.2 다른 기기에서 접속 (PC / 노트북 / 스마트폰)

1
서버 PC와 같은 네트워크(같은 공유기)에 연결합니다. 유선/WiFi 모두 가능합니다.
2
서버 PC의 앱 화면 상단 또는 타이틀바에 표시된 IP 주소를 확인합니다.
예: http://192.168.0.10:3000
3
브라우저(Chrome 권장)를 열고 주소창에 해당 주소를 입력합니다.
4
로그인 화면이 나타나면 계정 정보를 입력하여 접속합니다.
팁: 스마트폰 즐겨찾기 스마트폰 브라우저에서 접속 후 "홈 화면에 추가"를 하면 앱처럼 바로 실행할 수 있습니다.

5.3 시작프로그램 등록 (권장)

서버 PC를 켤 때마다 앱이 자동으로 실행되도록 시작프로그램에 등록하면, 별도 조작 없이 항상 시스템이 가동됩니다.

Windows

1
키보드에서 Win + R 키를 눌러 실행 창을 엽니다.
2
shell:startup 을 입력하고 확인을 누릅니다. 시작프로그램 폴더가 열립니다.
3
바탕화면의 HR SYSTEM 바로가기를 이 폴더에 복사합니다.
(바로가기를 마우스 오른쪽 클릭 → 복사 → 시작프로그램 폴더에 붙여넣기)
4
완료! 이제 PC를 켜면 HR SYSTEM이 자동으로 실행됩니다.
확인 방법 PC를 재시작한 후 작업표시줄(트레이)에 HR SYSTEM 아이콘이 나타나는지 확인합니다.

macOS

1
시스템 설정을 엽니다. (좌측 상단 Apple 메뉴 → 시스템 설정)
2
일반 → 로그인 항목 및 확장 프로그램을 클릭합니다.
3
"로그인 시 열기" 목록 아래의 + 버튼을 클릭합니다.
4
응용 프로그램 폴더에서 HR SYSTEM을 선택하고 열기를 클릭합니다.
5
완료! 이제 Mac을 켜면 HR SYSTEM이 자동으로 실행됩니다.
참고 시작프로그램 등록 후에도 서버 PC의 전원이 켜져 있어야 다른 기기에서 접속이 가능합니다. 절전 모드나 화면 잠금은 괜찮지만, PC가 꺼져 있으면 접속이 불가합니다.

6. 주요 기능 사용법

6.1 로그인 / 회원가입

로그인

  1. 이메일과 비밀번호를 입력합니다.
  2. 로그인 버튼을 클릭하면 대시보드로 이동합니다.
  3. 비밀번호를 잊은 경우 "비밀번호 찾기" 링크를 클릭합니다.

회원가입

  1. 로그인 화면에서 "회원가입" 링크를 클릭합니다.
  2. 이름, 이메일, 비밀번호, 전화번호, 부서, 직급을 입력합니다.
  3. 회원가입 버튼을 클릭하면 바로 로그인할 수 있습니다.

6.2 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 주요 현황을 한눈에 파악할 수 있습니다.

항목설명
잔여 휴가본인의 남은 휴가 일수
결재 대기처리해야 할 결재 건수 (관리자)
부서 인원소속 부서의 현재 인원 수
시간외근무이번 달 시간외근무 시간
오늘 근무출근/퇴근 시간, 현재 근무 상태
최근 휴가 신청최근 신청 내역 목록
결재 대기 목록승인이 필요한 건 목록 (관리자)

6.3 출퇴근 관리

오늘 근무 (출퇴근 기록)

  1. 좌측 메뉴에서 근태관리 → 오늘 근무를 클릭합니다.
  2. 출근 버튼을 클릭하면 현재 시간이 출근 시간으로 기록됩니다.
  3. 퇴근 시 퇴근 버튼을 클릭하면 퇴근 시간이 기록됩니다.
  4. 근무시간은 출근~퇴근 시간에서 점심시간을 제외하여 자동 계산됩니다.

내 근태현황

근태관리 → 내 근태현황 메뉴에서 월별 출퇴근 기록과 근무시간을 요약하여 확인할 수 있습니다.

연장근무 신청

  1. 근태관리 → 연장근무 신청 메뉴에서 연장근무 신청 버튼을 클릭합니다.
  2. 근무 유형(평일야간/주말/공휴일), 근무 일자, 시작/종료 시간, 사유를 입력합니다.
  3. 신청 버튼을 클릭하면 결재자에게 승인 요청이 전달됩니다.

6.4 휴가 관리

나의 휴가 조회 및 신청

  1. 좌측 메뉴에서 휴가 → 나의 휴가를 클릭합니다.
  2. 상단에서 휴가 유형별 잔여 일수를 확인합니다.
  3. 휴가 신청 버튼을 클릭합니다.
  4. 휴가 유형 선택 → 시작일/종료일 선택 → 사유 입력 후 신청 버튼을 클릭합니다.
  5. 결재선에 따라 자동으로 결재자에게 알림이 발송됩니다.
휴가 구분
  • 종일 — 하루 전체 (1일 차감)
  • 오전반차 — 오전만 사용 (0.5일 차감, 연차에서 차감)
  • 오후반차 — 오후만 사용 (0.5일 차감, 연차에서 차감)

휴가 사용 현황 (캘린더)

휴가 → 사용현황 메뉴에서 캘린더 탭을 선택하면, 부서원들의 휴가 일정을 달력 형태로 확인할 수 있습니다. 공휴일은 빨간색으로 표시됩니다.

6.5 복지 관리

복지 신청

  1. 좌측 메뉴에서 복지를 클릭합니다.
  2. 카테고리별로 이용 가능한 복지 항목이 표시됩니다.
  3. 원하는 항목을 클릭하면 신청 팝업이 나타납니다.
  4. 금액, 사유 등을 입력하고 신청 버튼을 클릭합니다.
자동승인 항목 도서구입비, 식대 등 자동승인으로 설정된 항목은 신청 즉시 승인 처리됩니다. 별도의 결재가 필요 없습니다.

6.6 결재 처리

결재 대기건 확인 및 처리

  1. 대시보드의 결재 대기 영역 또는 휴가 → 신청 관리 메뉴에서 결재 대기건을 확인합니다.
  2. 각 건의 상세 내용을 확인한 후 승인 또는 반려 버튼을 클릭합니다.
  3. 반려 시 사유를 입력할 수 있습니다.
결재 흐름
  • 순차 결재 — 부장 → 이사 → 대표 순서로 결재가 진행됩니다.
  • 선승인 — 상위 결재자(예: 대표)가 먼저 승인하면 하위 단계는 자동으로 건너뜁니다.
  • 반려 — 어느 단계에서든 반려하면 즉시 반려 처리됩니다.

시스템 관리자(SYSTEM_ADMIN) 또는 회사 관리자(COMPANY_ADMIN) 권한이 있는 경우 관리자 메뉴가 표시됩니다.

8. 자주 묻는 질문 (FAQ)

Q. 서버 PC를 꺼도 되나요?

A. 서버 PC가 꺼지면 다른 기기에서 접속이 불가능합니다. 근무 시간 동안은 서버 PC를 켜두어야 합니다. 5.3 시작프로그램 등록을 참고하여 자동 실행을 설정하면, PC를 켤 때마다 앱이 자동으로 시작되어 편리합니다.

Q. 여러 대에 앱을 설치하면 데이터가 공유되나요?

A. 아닙니다. 각 PC에 설치하면 별도의 독립된 시스템이 됩니다. 데이터를 공유하려면 한 대만 설치하고 나머지는 브라우저로 접속해야 합니다.

Q. 데이터 백업은 어떻게 하나요?

A. 서버 PC의 아래 경로에 있는 msa-hr.db 파일을 복사하면 됩니다.

  • Windows: C:\Users\[사용자명]\AppData\Roaming\msa-hr\msa-hr.db
  • macOS: ~/Library/Application Support/msa-hr/msa-hr.db

Q. IP 주소가 바뀌었어요.

A. 서버 PC의 IP가 변경되면 다른 기기에서 새 IP로 접속해야 합니다. IP 변경을 방지하려면 공유기에서 서버 PC에 고정 IP를 할당하는 것을 권장합니다.

Q. 외부(집, 외근지)에서 접속할 수 있나요?

A. 기본적으로 같은 네트워크 내에서만 접속 가능합니다. 외부 접속이 필요한 경우 VPN 또는 Cloudflare Tunnel 등 추가 설정이 필요합니다. 별도 문의해주세요.

Q. 동시 접속자 수 제한이 있나요?

A. 소프트웨어 상의 제한은 없습니다. 서버 PC의 성능에 따라 수십 명까지 동시 접속이 가능합니다.

9. 문제 해결

증상해결 방법
앱 실행 후 화면이 안 나옴잠시 기다려주세요 (최초 실행 시 10~20초 소요). 계속 안 되면 앱을 종료 후 재실행합니다.
다른 기기에서 접속 안 됨① 같은 네트워크인지 확인 ② 서버 PC 방화벽에서 3000번 포트 허용 ③ IP 주소가 정확한지 확인
로그인은 되는데 화면 안 바뀜브라우저 캐시 삭제 후 재접속 (Ctrl+Shift+Delete)
비밀번호를 잊었어요관리자에게 비밀번호 초기화를 요청합니다. 관리자는 직원 관리에서 비밀번호를 재설정할 수 있습니다.
macOS에서 "손상된 파일" 오류터미널에서 xattr -cr /Applications/HR\ SYSTEM.app 실행 후 다시 엽니다.
데이터가 사라졌어요앱 삭제/재설치 시 데이터가 초기화될 수 있습니다. 정기적인 DB 파일 백업을 권장합니다.
