1. Executive Summary

HR SYSTEM은 중소기업(10~200명)을 위한 온프레미스 인사관리 솔루션입니다. 별도의 서버 인프라나 클라우드 구독 없이, 사내 PC 한 대에 설치만으로 전사 인사관리 시스템을 구축할 수 있습니다.

2. 문제 정의

2.1 시장 현황 및 Pain Point

Pain Point현황영향
높은 도입 비용 클라우드 SaaS 인사시스템은 인당 월 5,000~30,000원의 구독료 발생. 50인 기업 기준 연 300~1,800만원 중소기업 예산으로 부담
복잡한 구축 온프레미스 ERP는 서버 구매, DB 설치, 네트워크 구성 등 IT 전문 인력 필요 IT 인력 없는 중소기업 도입 불가
엑셀 의존 여전히 많은 중소기업이 엑셀로 휴가/인사 관리. 파일 충돌, 버전 관리, 공유 문제 업무 비효율, 데이터 유실 위험
데이터 외부 유출 우려 클라우드 기반 서비스는 직원 개인정보가 외부 서버에 저장 개인정보보호법 우려

2.2 타겟 고객

Primary Target 직원 10~200명 규모의 중소기업 중 아래 조건에 해당하는 기업:
  • 전담 IT 인력이 없거나 제한적인 기업
  • 클라우드 구독 비용을 최소화하고 싶은 기업
  • 직원 개인정보를 사내에서 관리하고 싶은 기업
  • 엑셀 기반 인사관리에서 시스템화로 전환하고 싶은 기업

3. 솔루션 개요

3.1 핵심 가치 제안

#가치설명

2제로 월비용일회성 라이선스 비용만 발생. 월 구독료, 인당 과금 없음

4즉시 사용설치 후 5분 이내 전 직원이 브라우저로 접속하여 사용 가능
5올인원휴가, 복지, 결재, 직원관리를 하나의 시스템에서 통합 관리

7.2 백업 및 복구

백업 방법
  1. 서버 PC의 DB 파일을 정기적으로 USB 또는 네트워크 드라이브에 복사합니다.
    • Windows: %APPDATA%\msa-hr\msa-hr.db
    • macOS: ~/Library/Application Support/msa-hr/msa-hr.db
  2. 주 1회 이상 백업을 권장합니다.
  3. 복구 시 백업 파일을 동일 경로에 복사하면 됩니다.
